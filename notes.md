# Глоссарий базовых понятий веб-разработки (INTERN-66)

1. **IP-адрес** — уникальный числовой адрес компьютера или сервера в сети (например, 192.168.1.1), используемый для маршрутизации данных.
2. **Домен** — понятное человеку текстовое имя сайта (например, google.com), которое привязывается к числовому IP-адресу.
3. **DNS (Domain Name System)** — сетевая служба ("телефонная книга" интернета), которая переводит доменные имена в понятные компьютерам IP-адреса.
4. **HTTP/HTTPS** — протокол передачи данных, определяющий правила взаимодействия между клиентом (браузером) и сервером. HTTPS является защищенной версией с шифрованием данных.
5. **Frontend** — клиентская часть веб-приложения, с которой напрямую взаимодействует пользователь в браузере (состоит из HTML, CSS и JavaScript).
6. **Backend** — серверная часть приложения, отвечающая за логику работы, обработку данных, взаимодействие с базами данных и безопасность.
7. **Рендеринг** — процесс, при котором движок браузера обрабатывает полученный от сервера HTML/CSS/JS код и превращает его в визуальную веб-страницу.
